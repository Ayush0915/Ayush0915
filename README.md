## 🚀 Featured Projects

### [CodeVerdict](https://github.com/Ayush0915/CodeVerdict)
Multi-agent AI code review system — four specialist agents (Security, Quality, Performance, Test Coverage) plus a Synthesizer, built on FastAPI with Groq/Llama 3, FAISS-based RAG grounded in real security documentation, Bandit SAST integration, and asyncio parallel orchestration.
**87.5% precision · 100% recall** — [Live demo](https://code-verdictt.vercel.app)

### [Composio App Research Agent](https://github.com/Ayush0915/composio-app-research)
Agent pipeline that researches 100 apps across 10 categories, verifies accuracy through a two-pass loop, surfaces cross-app patterns, and outputs a self-contained HTML case study. Uses Groq (Llama 3.3 70B) and Composio-managed Tavily search/extract tools.

### [Cisco Forecast League (iNSIGHTX)](https://github.com/Ayush0915/cisco-cfl)
Multi-model ML forecasting pipeline (SARIMA, XGBoost, LightGBM, Prophet, hybrid residual-correction model) with a Streamlit dashboard and automated PDF report generation powered by Groq/Llama 3.

### Digital Wellbeing Intelligence System
ML pipeline for behavioral clustering and prediction — persistent cluster-label mapping, NumPy-based z-score outlier detection, and a JSON prediction API served via FastAPI.

### [Portfolio Website](https://github.com/Ayush0915/portfolio)
Personal site built with Next.js, TypeScript, and Tailwind CSS, featuring a 3D GitHub contributions globe and an AI chatbot. Live at [ayushkr-bhadani.vercel.app](https://ayushkr-bhadani.vercel.app).
