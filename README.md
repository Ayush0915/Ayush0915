**[CodeVerdict](https://github.com/Ayush0915/CodeVerdict)** — Multi-agent AI code review system
A RAG-powered, multi-agent workflow (Security, Quality, Performance, Test Coverage agents) that reviews pull requests using Llama 3, FAISS vector search, and static analysis (Bandit) — achieving 87.5% precision / 100% recall on a labeled benchmark.
`Python` `FastAPI` `Groq/Llama3` `FAISS` `RAG` `Agentic AI`

**[Digital Wellbeing Intelligence System](https://github.com/Ayush0915/digital-wellbeing-intelligence-system)** — Behavioral risk profiling via ML
An end-to-end ML pipeline that clusters users into behavioral risk profiles (K-Means, ~85% consistency) from 10K+ behavioral records, with automated Plotly dashboards and a FastAPI backend for real-time inference.
`Python` `scikit-learn` `Plotly` `FastAPI` `NLP`
